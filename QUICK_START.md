## RIGHT NOW: Follow These Steps

### Step 1: Create Repository on GitHub (2 min)

Go to https://github.com/new
- Name: `kenya-youth-dashboard`
- Public (not private)
- Click "Create repository"
- **Copy the URL** it gives you (looks like https://github.com/YOUR-USERNAME/kenya-youth-dashboard.git)

### Step 2: Open Terminal & Run These Commands (5 min)

Open Terminal/Command Prompt and navigate to your kenya-youth-dashboard folder:

```bash
cd /path/to/kenya-youth-dashboard
```

Then paste these commands **one by one**:

```bash
git init
```

```bash
git add .
```

```bash
git commit -m "Initial commit: Kenya Youth Opportunity Dashboard - 42 county analysis for youth employment, education, and digital access"
```

```bash
git remote add origin https://github.com/YOUR-USERNAME/kenya-youth-dashboard.git
```

(Replace YOUR-USERNAME with your actual GitHub username)

```bash
git branch -M main
```

```bash
git push -u origin main
```

**When it asks for credentials:** Use your GitHub username + personal access token (or password).

### Step 3: Enable GitHub Pages (2 min)

Go to: https://github.com/YOUR-USERNAME/kenya-youth-dashboard/settings/pages

- Source: "Deploy from a branch"
- Branch: main
- Folder: root
- Click Save

Wait 1-2 minutes.

### Step 4: Your Dashboard is Live! 🎉

**Visit:** https://YOUR-USERNAME.github.io/kenya-youth-dashboard/

Save this URL — you'll share it with Mastercard Foundation.
